# unity关于提取带有动画的模型的动画
# 准备工作
1. 一个带有动作的fbx模型
# 步骤
1. 将带有动作的模型导入unity,新建一个动画文件
2. 双击模型预制体的动画,打开动画窗口,全选动画关键帧
![图片](/markdown_img/unity/Pasted_image_20260507211117.png)
3. ctrl+c复制,在打开新的动画文件ctrl+a粘贴解决
